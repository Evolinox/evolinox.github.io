# Zmigrod Testcenter Scenarios

Scenario Collection to learn the Basics of Locomotives in SimRail

![PKP Cargo ET25 002](/assets/zmigrod_et25.png)

## Features

- Includes Scenarios for each Locomotive in SimRail
- Learn how to start a Locomotive
- Learn how to drive a Locomotive
- Also includes ETCS runs for ED250 and ET25

## Download

You can get these Scenarios from the [GitHub release page](https://github.com/Evolinox/SR_ZmigrodTestcenter/releases).
